 # Web220810_博客前端
--------  
前端使用 Angular     

后段代码：[https://github.com/TheStrongmans/Web220810/tree/master](https://github.com/TheStrongmans/Web220810/tree/master)  

运行项目  
1. 安装node.js  
2. 安装angular-cli  
win+r -> cmd -> 运行  
```
npm install -g typescript
```
``` 
cnpm i -g @angular/cli
```  
3. cd 至项目文件夹  
1) 下载项目依赖  
```
npm insert
```  
2)  运行项目  
```
ng serve
```

运行号后可在浏览器输入`http://localhost:4200/` 访问项目
